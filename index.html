About the work.

whoami

I’ve spent the last five years operating production infrastructure end-to-end — from the boring fundamentals of stateful servers to running multi-cluster Kubernetes on self-managed RKE2, Amazon EKS, and Azure AKS, with ArgoCD doing GitOps and KEDA handling event-driven autoscaling.

Lately a big part of my focus has been AI-augmented operations. I built an internal MCP server with FastMCP that exposes live system state to AI assistants, plus a multi-agent orchestration pipeline using Claude Code subagents. It changes the texture of incident response in ways I’m still figuring out.

I care about reliability as a discipline, not a job title. Good automation removes toil; bad automation creates new toil dressed in a YAML wrapper. I try to build the first kind.
